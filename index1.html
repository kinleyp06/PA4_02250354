My Major Project

Project Title: Fly Hi - Student Performance & Motivation App
Timeline: 2023 | UNICEF Bhutan Women Hackathon (Runner-Up)
My Role: UI/UX Designer & Wireframer (Team Project)
The Goal: Bridge the gap between teacher feedback and student motivation by designing an intuitive educational platform.

My Contribution: Led the UI/UX design process, translating the core concept into a functional and user-friendly prototype.
Wireframing: Created the initial application flow and screen layouts, establishing the user journey for both students and teachers.
UI Design: Designed the visual interface, focusing on an engaging and clear aesthetic to enhance the user experience.

Other Workshops & Programs

Python Programming Fundamentals: Strengthened my core programming logic and problem-solving skills.
pi-top [4] Exploration: Gained hands-on experience integrating hardware with software, understanding the basics of robotics and IoT.

Key Features I Designed:

A Performance Analytics Dashboard for students to visually track their progress.
The Teacher Feedback System interface, enabling personalized communication.
The Gamification & Rewards display for certificates and achievements.
The Study Planner module with integrated reminders.

The Outcome: Our user-centric design was a key factor in the project being awarded Runner-Up, demonstrating the value of a well-structured and intuitive interface.
What I Learned: The entire product design lifecycle—from initial wireframes to a high-fidelity prototype—and how effective design is crucial for user adoption and a project's success.
